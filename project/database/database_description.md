# Описание базы данных проекта "Доставку новогодних подарков"

## Таблицы

### auth_user (кастомная модель User)
Хранит учётные записи всех пользователей системы (дети, операторы, курьеры, сотрудники организаций).

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| id | integer | да | Первичный ключ |
| username | varchar(150) | да | Имя пользователя, уникальное |
| password | varchar(128) | да | Хэш пароля |
| email | varchar(254) | нет | Электронная почта |
| first_name | varchar(150) | нет | Имя |
| last_name | varchar(150) | нет | Фамилия |
| is_active | boolean | да | Активен ли пользователь |
| is_staff | boolean | да | Доступ в админку |
| date_joined | datetime | да | Дата регистрации |

### category
Категории подарков.

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| id | integer | да | Первичный ключ |
| name | varchar(100) | да | Название категории, уникальное |
| description | text | нет | Описание категории |

### item
Подарки (желания детей).

| Поле | Тип | Обязательное | Описание |
|------|-----|--------------|----------|
| id | integer | да | Первичный ключ |
| title | varchar(200) | да | Название подарка |
| description | text | нет | Описание |
| status | varchar(20) | да | Статус: draft, sent, approved, processing, delivered |
| category_id | integer | нет | Внешний ключ на category.id |
| created_at | datetime | да | Дата создания |
| updated_at | datetime | да | Дата последнего изменения |
| created_by_id | integer | нет | Внешний ключ на auth_user.id |

## Связи

- `item.category_id` → `category.id` (многие к одному) — каждый подарок относится к одной категории, в категории может быть много подарков.
- `item.created_by_id` → `auth_user.id` (многие к одному) — каждый подарок создан одним пользователем, пользователь может создать много подарков.
