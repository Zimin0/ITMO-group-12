# Сценарий видеодемонстрации Santa CRM (до 4 минут)

## Подготовка перед записью
1. Запустить сервер: `cd project && python manage.py runserver`
2. Убедиться, что в БД есть тестовые данные (категории, подарки, пользователи разных ролей)
3. Открыть браузер на `http://127.0.0.1:8000/`
4. Включить запись экрана

---

## Часть 1: Регистрация и вход (30 сек)

1. **Показать главную страницу** — объяснить, что это система для неавторизованного пользователя
2. **Нажать «Регистрация»** → заполнить форму:
   - Логин: `test_child`
   - Пароль: любой
   - Роль: **Ребёнок**
   - Телефон, адрес — заполнить
3. **Нажать «Зарегистрироваться»** → показать автоматический вход и главную страницу с ролью «Ребёнок»

---

## Часть 2: Сценарий «Ребёнок» (30 сек)

1. **Нажать «Написать письмо Дедушке»**
2. **Заполнить текст письма**: «Дорогой Дедушка Мороз, хочу конструктор LEGO!»
3. **Нажать «Отправить»** → показать сообщение об успехе
4. **Перейти в «Мои письма»** → показать письмо со статусом «Новое»

---

## Часть 3: Сценарий «Помощник Деда Мороза» (30 сек)

1. **Выйти** из учётной записи ребёнка
2. **Войти** как помощник (заранее созданный аккаунт)
3. **Открыть список писем** → показать все письма, включая новое
4. **Нажать на письмо** → прочитать содержание
5. **Нажать «Одобрить»** → показать изменение статуса

---

## Часть 4: Сценарий «Складская служба» (30 сек)

1. **Войти** как складской работник
2. **Открыть «Управление складом»** → показать список товаров с количеством
3. **Обновить количество** одного из товаров (например, с 10 на 15)
4. **Нажать «Обновить»** → показать обновлённое значение

---

## Часть 5: Сценарий «Курьер» (30 сек)

1. **Войти** как курьер
2. **Открыть «Маршруты доставки»** → показать назначенные доставки
3. **Обновить статус** одной доставки на «В пути»
4. **Обновить статус** на «Доставлено» → показать фиксацию даты

---

## Часть 6: Отчётность и PDF (30 сек)

1. **Войти** как администратор (или помощник)
2. **Открыть «Отчёты»** → показать дашборд со статистикой:
   - Количество писем, заказов, доставок
   - Разбивка по статусам
   - Статистика по категориям
3. **Нажать «Экспорт PDF»** → скачать и открыть PDF-файл

---

## Часть 7: Django Admin (30 сек)

1. **Войти** как администратор (superuser)
2. **Открыть `/admin/`** → показать панель Django Admin
3. **Показать список моделей**: Users, Letters, Items, Categories, Orders, Deliveries, Inventory
4. **Открыть одну запись** → показать CRUD-возможности

---

## Завершение

- Остановить запись
- Общая длительность: ~3.5 минуты
- При записи можно пропустить Django Admin если не укладываетесь в 4 минуты
